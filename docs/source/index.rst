CLEWs-EU's documentation!
==============================
This repository is the online documentation for the CLEWS-EU model developed for the Horizon DIAMOND project. The documentation gives a brief overview on the model structure, the individual modules and their design, the data and assumptions that go into the developmeent of each module.
